1. ¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

La etiqueta principal utilizada para crear una tabla en HTML es la etiqueta "table". Esta etiqueta define el inicio y el fin de la tabla,
y dentro de ella se colocan las filas y las celdas de la tabla utilizando las etiquetas "tr" y "td" respectivamente.

2. ¿Qué etiqueta se utiliza para definir una fila en una tabla?

La etiqueta utilizada para definir una fila en una tabla es la etiqueta "tr".

3. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

La etiqueta utilizada para crear una celda de encabezado en una tabla es la etiqueta "th". Esta etiqueta se utiliza para resaltar y dar formato
a las celdas que contienen encabezados de columna o fila en una tabla.

4. ¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

La etiqueta utilizada para crear una celda de datos en una tabla es la etiqueta "td". Esta etiqueta se utiliza para definir una celda regular
que contiene datos en una tabla.

5. ¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

El atributo que se utiliza para especificar cuántas columnas debe abarcar una celda en una tabla es "colspan". Puedes usar colspan con un valor númerico para indicar cuántas columnas adyacentes debe fusionar la celda en cuestión.

6. ¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

El atributo que se utiliza para especificar cuántas filas debe abarcar una celda es "rowspan". Puedes establecer el valor de este atributo en el
elemento "td" o "th" para indicar cuántas debería ocupar esa celda.

7. Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

Para crear una tabla con 10 filas debes usar la etiqueta "tr"(que define una fila) y colocarla 10 veces, una vez por cada fila que deseas en la tabla
